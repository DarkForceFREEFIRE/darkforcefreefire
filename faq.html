What is Rectify11?

Windows 11, by default has inconsistencies on it's UI elements (especially in dark mode). Rectify11 is a piece of software that aims for a more consistent Windows 11 experience.


Is Rectify11 safe to use?

Yes, it is not malware and won't affect performance as long as your apps work fine.

Detailed explanation: Please make sure you download Rectify11 from its official sources (https://rectify11.net/, https://github.com/Rectify11/Installer/releases, https://github.com/Rectify11/Installer/actions)

ONLY and not from some random link.

NOTE: SecureUX ThemeTool is commonly flagged by antiviruses which Rectify11 uses

Make sure to turn off all antiviruses before installing Rectify11

Please read the "Some applications do not launch." section too.

We only support official Windows images, so please do not use a "debloated" ISO as those cause more problems than they fix.


Will Rectify11 have an ISO image after version 2.5?

No, Rectify11 uses an installer that you have to install on top of your existing Windows installation.


Will I lose my data after installing Rectify11?

No, unless you reinstalled Windows


What are the minimum system requirements to run Rectify11?

  • Windows 11 to Windows 11 24H2 is currently supported. Note that context menus may be screwed up if you select Nilesoft shell in the installer.


Some applications do not launch.

For some reason, some software does not like Rectify11 modifying icons. For ESET antivirus, uncheck msi.dll icon in the installer. It is also said that reWASD does not work for some reason.


My themes look messed up.

There are multiple reasons for this problem.


How to remove white and colored borders from File explorer or control panel.

Open the Rectify11 installer, and patch shell32 and explorerframe

My theme looks like a mess after pressing CTRL+ALT+DEL or when locking my PC.

Download SecureUX ThemeTool and open it. There, look at the "Patcher" category and make sure "LogonUI hooked" says Yes. If it says No, click Uninstall on the "Installation" category and make sure "Rename DefaultColors" and "Hook LogonUI" are checked, then press "Install" and restart.

My apps look like a mess.

Find MicaForEveryone's icon in your system tray and open it. A settings window should open. Click the + icon, choose "Add process Rule", choose your app's executable file, then press "Add". A new option with your application's name will appear on the sidebar. Click that and set "Backdrop Type" to "None" and disable "Extend Frame into Client Area".

If that doesn't work, open the Rectify11 control panel. Choose either Rectify11 Partial dark theme or Rectify11 light theme in the list. Press apply.

If that also didn't work, most likely the developer of the application hardcoded background color or text color. Contact the application developer.

My websites look messed up.

Mozilla Firefox: Go to Settings. In the General tab of the page, find "Override Firefox’s default colors for text, website backgrounds, and links" text, click "Manage Colors". The "Colors" window will open. Uncheck "Use system colors" and press OK. Other browsers: Install the Stylus extension, and then apply this userstyle


Mica does not work

There can be multiple causes for this problem.


Make sure your settings are correct.

Make sure that "Transparency effects" on Windows Settings (under Personalization > Colors) is turned on. After that, find MicaForEveryone's icon in your system tray and open it. A settings window should open. There, find "Global Rule" and open it. Make sure "Extend Client into Frame Area" is enabled and "Backdrop Type" is set to "Mica". Repeat for "explorer" as well.


The Mica colors look different from my "wallpaper".

It is because you are using Lively Wallpaper/Wallpaper Engine or other wallpaper apps. For this, you either have to disable these apps or change the wallpaper to be similar to your Windows desktop wallpaper (under Personalization > Wallpaper in the Windows Settings app).

Mica effects don't work properly on File Explorer.

This problem only happens on Windows 11, version 23H2 and above. The reason is some parts on the new File Explorer like the toolbar, home page, gallery are all XAML-based. MicaForEveryone is not designed to hook into XAML components and change the backdrop type. However, you can use ExplorerBlurMica


I don't like changes that Rectify11 did to my explorer, how to disable them and get rid of the hightlight border.

Don't select themes while installing/uninstall them. Also deselect ExplorerFrame.dll.mun from icons section.


Gadgets are not working after installation and the gadget picker doesn't have any gadgets to choose from.

Remove gadgets from control panel, and reinstall them from this link: https://github.com/Rectify11/Gadgets/releases/tag/ESR1
